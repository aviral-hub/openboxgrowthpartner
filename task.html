A Data Dictionary sets up a shared vocabulary, mapping every field label to technical definitions, intent contexts, and stakeholder access groups.

Column Key Business Description Operational Use Case Context

Part 4 — Identify Foreign Keys

Foreign Keys (FK) establish paths that connect separate relational matrices. By placing another table's Primary Key inside your transaction tracker, you link those datasets together cleanly.

Foreign Key Field References Parent Origin System Connection Intent
Customer ID Customers Table (PK) Tracks exactly which specific customer placed the transaction row.
Product ID Products Table (PK) Tracks exactly what coffee item SKU variant was bought.

Key Performance Indicator (KPI) Business Meaning & Operational Value

Monthly Sales Trend The timeline vector tracking performance over time. Uncovers demand patterns and seasonality.

Average Order Value measures purchasing concentration behavior. Compute this by dividing gross yield by your unique transaction record counts:

AOV = Total Revenue ÷ Total Orders
